# 💰 Expense Tracker & Categorizer

A simple web application that allows users to track their expenses and organize them into categories. This project was built to practice JavaScript functionality, data handling, and user interaction in a real-world scenario.

## 🚀 Features
- Add and track expenses with descriptions and amounts  
- Categorize expenses (e.g., Food, Transportation, Entertainment)  
- Display a list of recorded expenses  
- Calculate total spending  
- Simple and user-friendly interface  

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

## 💡 Purpose
This project was created to strengthen my understanding of JavaScript and front-end development by building a practical tool for managing personal expenses. It focuses on handling user input, organizing data, and updating the interface dynamically.

## 📚 What I Learned
- Working with JavaScript to handle user input and events  
- Updating the DOM dynamically based on data  
- Structuring and organizing data using arrays and objects  
- Improving problem-solving and debugging skills  

## 🔧 Future Improvements
- Add local storage to save expenses  
- Create charts or visual summaries of spending  
- Add filters by category or date  
- Improve UI/UX design
